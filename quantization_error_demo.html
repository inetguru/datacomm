Linear Quantization
The amplitude range [−A, +A] is divided into L = 2ᵏ equal intervals. Each sample is mapped to the nearest level midpoint. The step size is:
Δ = 2A / 2ᵏ = A / 2^(k−1)
Quantization Error
The error is bounded by ±Δ/2. For uniform quantization, the error is approximately uniformly distributed, giving an RMS error of:
RMSE ≈ Δ / √12 = Δ / 2√3
SNR & Information
Each extra bit of depth halves the step size, reducing RMSE by 6 dB and doubling the data rate. The theoretical SNR for a sinusoidal input is:
SNR ≈ 6.02k + 1.76 dB
